### * commit rules
~~~zsh
    git commit -m 'Add - 내용 (#)' : 새로운 것을 추가
    git commit -m 'Update - 내용 (#)' : 기존에서 기능 추가
    git commit -m 'Fix - 내용 (#)' : 고침
    git commit -m 'Modify - 내용 (#)' : 수정
~~~
